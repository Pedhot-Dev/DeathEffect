# DeathEffect
DeathEffect Plugin for PocketMine

# Concept
Spawn effect on players when they die in game at their coordinates

# Effect list
| Name  | Cost | Description |
| ------------- | ------------- | ------------- |
| Default  | 0  | None effect
| Lightning  | 100000  | Lightning strike |
| More...  | More...  | More... |
